# 기능 목록

1. 경주할 자동차 이름 입력
2. 시도할 횟수 입력
3. 실행 결과 표시
4. 게임 우승자 표시
5. 에러(예외 상황) 표시

- 이름 입력이 잘못된 경우
  - 쉼표가 없는 경우
  - 중복된 이름인 경우(대문자, 소문자 체크)
  - 이름 길이가 5자 초과인 경우, 2자 미만인 경우
  - 자동차 개수가 20개 초과인 경우
- 시도 횟수가 잘못된 경우
  - 시도 횟수 20번 초과한 경우
  - 자연수가 아닌(음수인) 경우
  - 문자인 경우
